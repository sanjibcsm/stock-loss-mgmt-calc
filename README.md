# stock-loss-mgmt-calc
- This application is created to calculate how much share you need to create to balance your share market loss.
- Created with helpof AI (Amazon Q)
- Suppose you have 10 shares of a XYZ company with buying price 267.
-   And  current market price is 235 now.
-   So you have loss now 267-235=32 each * 10 = 320
-   If you want to buy new shares with current market price i.e. 235, then much shares need to buy so that your average price will became equal to current market price.
